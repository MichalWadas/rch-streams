## [0.3] - 2021-08-13

### New/Improved

-   Separate data creation in Mambu for each installation/runtime
-   Added new user Emily in LE json
-   Updated Mambu Service to create unnecessary clients in Mambu
-   Client's email, phone and home address are updated in Mambu with values from legal-entity-hierarchy.json
-   Code refactoring to remove code warning
-   Uploading account statement as part of bootstrap task when creating new account for any client

### What has Changed

-   With new bootstrap task run, Username in identity and mambu will be <installation-name>-<runtime-name>-externalId i.e. ref-dev-paolo

### Bug Fixes

-   Fixed the mismatch b/w externalId and BBAN of accounts in LE json
-   Fixed create client api call previously failing due to null client id
